## 전체적인 구상

C#과 WPF를 이용해서 GUI를 만들고 내부적으로는 C++을 이용해서 렌더링 및 기타 핵심 작업을 수행할 예정입니다.

## OBJ 파일 포맷

OBJ 파일은 3D 모델링 데이터를 담은 파일 형식입니다.  
텍스트로 이루어져 있으며 각 줄마다 데이터를 하나씩 담고있습니다.
각 줄의 첫번째로 오는 문자에 따라 그 데이터가 정점인지 정점의 노말벡터인지 텍스쳐인지 면인지 결정됩니다.

| 문자 | 의미            |
| ---- | --------------- |
| v    | 정점            |
| vn   | 정점의 노말벡터 |
| vt   | 텍스쳐          |
| f    | 면              |
